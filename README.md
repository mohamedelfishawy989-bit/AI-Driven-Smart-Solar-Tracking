# AI-Driven-Smart-Solar-Tracking
Embedded Systems Engineer &amp; IoT Developer

Architected a dual-axis solar tracking system utilizing a quad-sensor LDR array, optimizing panel alignment to maximize energy harvest by approximately 30% compared to static systems.
Engineered an autonomous thermal management and self-cleaning mechanism  mitigating efficiency loss due to dust accumulation and overheating.
Implemented Edge AI algorithms on microcontrollers to process sensor data locally, reducing latency and ensuring system reliability without cloud dependency.
Developed a real-time IoT web dashboard featuring live telemetry and remote actuation capabilities for system overrides.
Integrated power monitoring sensors to calculate real-time efficiency metrics, enabling data-driven decisions for system optimization.
